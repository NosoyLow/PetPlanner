# PetPlanner - PIA Aplicaciones Móviles
## Requisitos
**Tener instalado:**
- Node.js
- Angular
- Ionic Framework
- Android Studio
## Pasos para utilizar la app
- Clonar el repositorio
- Abrir la consola y poner los siguientes comandos:
- `npm install`
- `ionic build`
- `ionic cap copy`
- `ionic cap sync`
- `ionic cap open android`
- Exportar el proyecto de Android Studio como apk
